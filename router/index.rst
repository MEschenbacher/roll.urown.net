Firewall-Router
===============

.. image:: OpenWrt-logo.*
    :alt: OpenWrt Logo
    :align: right

`OpenWrt <https://openwrt.org/>`_ is an embedded operating system based on the
Linux kernel, and primarily used on embedded devices to route network traffic.
All components have been optimized for size, to be small enough for fitting into
the limited storage and memory available in home routers.
